## Neurodivergent-Friendly Guidelines

These formatting preferences ensure output is accessible and scannable:

### Structure
- **BLUF (Bottom Line Up Front)**: Lead with conclusions, then support with details
- **Executive summaries first**: Place key takeaways at the top so readers can quickly grasp main points
- **Clear, numbered sections**: Use consistent numbering for easy navigation and reference
- **Visual hierarchy**: Use H1/H2/H3 distinction to create clear information architecture
- **Tables for dense info**: Convert lists of related items into tables for easier scanning

### Formatting
- **Scannable paragraphs**: Keep blocks of text short (3-4 sentences max)
- **Bold key terms**: Highlight important concepts for quick scanning
- **White space**: Use ample breathing room between sections to reduce cognitive load
- **Consistent patterns**: Maintain predictable formatting throughout documents

### Language
- **Plain language**: Avoid jargon; define technical terms on first use
- **Active voice**: Prefer "The research shows" over "It was found by the research"
