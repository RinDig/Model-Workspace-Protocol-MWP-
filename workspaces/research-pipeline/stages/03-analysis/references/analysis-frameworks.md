# Analysis Frameworks

Categorization patterns for extracting insights.

## Finding Types
| Type | Marker | Example |
|------|--------|---------|
| Fact | `[FACT]` | Statistics, dates, verifiable claims |
| Opinion | `[OPINION]` | Author views, predictions, judgments |
| Method | `[METHOD]` | Processes, techniques, approaches |
| Finding | `[FINDING]` | Research results, discovered patterns |
| Claim | `[CLAIM]` | Unverified assertions requiring evidence |

## Evidence Strength
| Strength | Mark When |
|----------|-----------|
| Strong | Primary data, peer-reviewed, reproducible |
| Moderate | Secondary sources, expert opinion, industry report |
| Weak | Anecdotal, blog posts, unverified claims |

## Pattern Detection
- **Convergence:** Multiple sources agree on same point
- **Divergence:** Sources disagree on same point
- **Gap:** No sources address key question
- **Novelty:** Unique finding from single source

## Contradiction Types
| Type | Resolution |
|------|------------|
| Direct | A says X, B says not-X | Credibility-weighted |
| Partial | A says X, B says X sometimes | Nuanced synthesis |
| Contextual | A says X in context C1, B says not-X in C2 | Context-dependent |
| Temporal | A says X in 2020, B says not-X in 2024 | Timeline-dependent |
