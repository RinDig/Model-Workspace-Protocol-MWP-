# Output Templates

Report formats for different deliverable types.

## Standard Report Template
```markdown
# [Research Topic]

*Report generated: YYYY-MM-DD*
*Methodology: [Methodology used]*
*Sources: [N] (Avg credibility: [score])*

## Executive Summary
[2-3 paragraph overview with key findings and recommendations]

## Introduction
[Research question, context, and scope]

## Methodology
[How the research was conducted]

## Findings
### [Question 1 / Theme 1]
[Synthesized findings with citations]
**Confidence:** [High/Medium/Low]

### [Question 2 / Theme 2]
[Synthesized findings with citations]

## Conclusions
[Key conclusions drawn from findings]

## Limitations
[Gaps, contradictions, or uncertainties]

## Recommendations
[If applicable, actionable next steps]
```

## Metadata Requirements
| Field | Value |
|-------|-------|
| Generated date | Current date |
| Methodology | From 01-scoping |
| Source count | From 02-discovery |
| Avg credibility | Calculated from sources |

## Citation Formats
| Style | In-text | Reference |
|-------|---------|-----------|
| APA | (Author, Year) | Author, A. (Year). Title. Source. |
| Chicago | [n] | Author, A. Title. Source, Year. |
