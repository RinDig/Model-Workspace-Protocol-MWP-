# Research Pipeline

Guide any research project from initial question to final deliverable.

## Task Routing

| Task Type | Go To | Description |
|-----------|-------|-------------|
| Start new research | `stages/01-scoping/CONTEXT.md` | Define goal, scope, and methodology |
| Gather information | `stages/02-discovery/CONTEXT.md` | Search and collect sources |
| Analyze findings | `stages/03-analysis/CONTEXT.md` | Extract insights and patterns |
| Draw conclusions | `stages/04-synthesis/CONTEXT.md` | Combine findings into arguments |
| Format deliverable | `stages/05-output/CONTEXT.md` | Generate report, dataset, or docs |

## Triggers

| Keyword | Action |
|---------|--------|
| `research [topic]` | Run full pipeline from scoping through output |
| `setup` | Run onboarding questionnaire |
| `status` | Show pipeline completion for all stages |
| `resume` | Read PROGRESS.md and continue from last checkpoint |
